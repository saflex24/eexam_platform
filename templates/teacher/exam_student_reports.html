How to Print Student Reports
Printing Individual Reports:
  1. Click the button next to any student
  2. The report will download as a Word document (.docx)
  3. Open the downloaded file in Microsoft Word
  4. Click File → Print or press Ctrl+P (Cmd+P on Mac)
  5. Adjust print settings as needed and print
Printing Multiple Reports:
  1. Select students using the checkboxes
  2. Click "Download Selected" or "Download All Reports (ZIP)"
  3. Extract the ZIP file to a folder
  4. Open each document and print, or:
    • Windows: Select all files → Right-click → Print
    • Mac: Select all files → File → Print Documents
Customizing School Info:

To add your school name and logo to reports:

  1. Open routes/teacher.py
  2. Find the get_school_settings() function
  3. Update the school information: 
    return {
    'school_name': 'Your School Name',
    'school_address': 'Your Address',
    'school_phone': 'Your Phone',
    'school_email': 'your@email.com',
    'school_logo_path': 'static/uploads/school_logo.png'
}
  4. Save your school logo as static/uploads/school_logo.png
  5. Restart the application
Tips:
  • Each report is designed to fit within 4 pages
  • Reports include detailed question analysis
  • All options are clearly marked (correct/incorrect)
  • Use "Print Selection" in Word to print specific pages
  • Consider duplex (double-sided) printing to save paper
